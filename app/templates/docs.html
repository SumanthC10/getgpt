About GetGPT

GetGPT (Genetic variants, differentially Expressed genes, and drug Targets) is an interactive platform that unifies disease-gene analysis into a single pipeline to accelerate hypothesis generation and drug discovery. The project is led by Sumanth Chandrupatla, Andrew Trotter, Kevin Song, and Jake Chen.

Core Functionality

The platform's primary purpose is to generate comprehensive gene lists for a given disease by aggregating data from multiple sources to provide a holistic view of disease genetics.

Gene List Creation

GetGPT generates integrated gene lists by aggregating data from multiple leading databases:

  • GWAS Catalog: The platform retrieves genetic associations from the GWAS Catalog.
  • RummaGEO: It identifies relevant gene sets from the Gene Expression Omnibus (GEO) using RummaGEO.
  • OpenTargets: It compiles a list of drug targets associated with the disease from OpenTargets.

Semantic Search

The platform uses advanced semantic search capabilities to help researchers find the most relevant EFO (Experimental Factor Ontology) terms for their queries, ensuring that the subsequent gene list generation is as accurate as possible.
